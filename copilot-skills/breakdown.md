# Issue Breakdown Skill

Break large issues into smaller, independently shippable sub-issues.

## Usage
- "Break down issue #42"
- "Split this into smaller tasks"
- "Plan sub-issues for #42"

## Tools
```bash
gh planning breakdown {issue_number} --repo {owner/repo} --dry-run --json
gh planning breakdown {issue_number} --repo {owner/repo} --json
```
