# Onboarding & Discovery

Help users learn gh-planning and discover commands.

## When to use

- User asks "how do I use gh-planning?"
- User asks "what can gh-planning do?"
- User wants to learn a workflow or find a command
- User is new to gh-planning and needs guidance
- User asks for a cheatsheet, tutorial, or guide

## Tools

### Interactive tutorial
Walk through gh-planning step by step:
```
gh planning tutorial
```

Resume where you left off, or jump to a specific lesson:
```
gh planning tutorial focus-mode
gh planning tutorial --list
gh planning tutorial --reset
```

Lessons: command-center, board-status, focus-mode, track-claim, team-standup, advanced-power

### Cheatsheet
Browse commands organized by scenario:
```
gh planning cheatsheet
gh planning cheatsheet --plain
```

Categories: Morning Routine, Starting a Task, During Work, Collaboration, AI & Agents, Wrapping Up, Configuration

### Workflow guides
Deep-dive walkthroughs for specific workflows:
```
gh planning guide morning
gh planning guide new-task
gh planning guide one-on-one
gh planning guide agent
gh planning guide breakdown
```

### General help
```
gh planning --help
gh planning <command> --help
```
